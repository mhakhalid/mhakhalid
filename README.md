## What I'm building this year

I run a personal AI agent system that scouts developer pain points across Reddit, GitHub, and Hacker News — then helps me design, build, and ship solutions. Every project below started as a real problem someone was frustrated about.

### 🔍 [llm-cost-tracker](https://github.com/mhakhalid/llm-cost-tracker)
> Real-time dashboard for tracking Claude / OpenAI / Gemini API spend across projects

Teams using LLM APIs have no easy way to see costs broken down by project, model, or user. Built a full-stack solution with an Express proxy that intercepts every API call, calculates real-time cost, and surfaces it on a React dashboard — with Slack alerts when budgets are hit.

`React` `Node.js` `Express` `Python` `Chart.js` `REST API`

---

### 🛠️ [devboard](https://github.com/mhakhalid/devboard)
> Personal dev command center — GitHub stats, open PRs, daily tasks, and streak tracker in one UI

Built this because I wanted one place to see everything about my dev workflow without jumping between tabs. Pulls live data from the GitHub API, displays contribution streak, open PRs, and a daily task queue. One-click deploy on Vercel.

`React` `GitHub API` `Vercel` `Node.js`

---

### 🤖 [agent-workflow-templates](https://github.com/mhakhalid/agent-workflow-templates)
> Open-source n8n + Claude workflow templates for common developer automations

Reusable automation workflow templates anyone can drop into their own n8n setup. Covers GitHub automation, scouting agents, content pipelines, and API cost tracking. Each template ships with a detailed README and a working JSON file.

`n8n` `Claude API` `Python` `Automation` `Open Source`

---

### 📡 [reddit-signal](https://github.com/mhakhalid/reddit-signal)
> CLI + API that scans subreddits for trending pain points and surfaces buildable project ideas

The tool I use to find what to build next. Give it a list of subreddits, it reads trending posts, runs them through Claude, and returns a ranked list of real problems worth solving. Includes a FastAPI wrapper so it runs as a standalone service or plugs into larger workflows.

`Python` `FastAPI` `Reddit API` `Claude API` `CLI`
